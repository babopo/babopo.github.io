AI Context Clipper Privacy Policy

Last updated: 2026-06-14

AI Context Clipper is a local-first Chrome extension that helps users collect selected text and web page content into source-backed prompt context for AI chat tools.

Data We Store

AI Context Clipper may store the following data locally in your browser:

  • Clipped page text
  • Selected text
  • Page titles
  • Page URLs
  • Capture timestamps
  • Source type metadata, such as page or selection
  • The current context bundle and recent clips

This data is stored with chrome.storage.local.

Data We Do Not Collect

AI Context Clipper does not collect, sell, transmit, or share user data.

The extension does not use:

  • User accounts
  • Remote servers
  • Cloud sync
  • Analytics
  • Advertising trackers
  • Built-in AI API calls
  • Third-party data processors

Clipboard Use

When you click a copy button, AI Context Clipper writes the selected prompt format to your clipboard. You decide where to paste that content.

Page Access

AI Context Clipper reads page content only for the product purpose of creating a local context bundle for AI prompts. It stores content only when you intentionally use the extension, such as:

  • Selecting text and choosing Add to AI Context
  • Clicking Clip page in the extension popup

The extension uses the Chrome activeTab and scripting permissions to access the active tab after a user action.

The extension also runs a content script so it can show a small action bubble after text selection on supported web pages. This bubble can be disabled in the extension settings. The content script does not send page content to any server.

Data Removal

You can remove stored data by:

  • Deleting individual clips
  • Clearing the current bundle
  • Removing the extension from Chrome
  • Clearing Chrome extension storage
